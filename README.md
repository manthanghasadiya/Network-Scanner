# Network-Scanner

This is a **network scanner** tool. This tool will scan and print the **IP address** and **MAC address** of all device which are connected on the **same network**. Or you can just find the **MAC address** of any target **IP address**.

<h2>How to USE?</h2>

Open terminal where you downloaded this tool and type:

  >**_python3 NetworkScanner.py -t [IP Address]_**
  
Check **_python3 NetworkScanner.py --help_** for more details.
